Dossier des images du site Hector AI
====================================

Déposez ici vos images. Fichiers attendus par le site :

- logo.svg               → Logo Hector (affiché dans le header, le footer et le hero).
                           Format SVG recommandé. Tant qu'il est absent, le site
                           affiche automatiquement le texte « Hector » à la place.

- hero.jpg               → Photo de fond plein écran de la page d'accueil.
                           Format paysage, haute résolution (ex. 1920x1080 ou plus).

- poster-analyse.jpg     → (Optionnel) Image d'aperçu de la vidéo « Analyse des
                           dossiers » sur la page « Comment ça marche ».

- poster-redaction.jpg   → (Optionnel) Image d'aperçu de la vidéo « Rédaction
                           d'actes » sur la page « Comment ça marche ».

- equipe/                → (Optionnel) Sous-dossier pour les photos de l'équipe
                           (voir les commentaires dans equipe.html).

Ce fichier README.txt peut être supprimé : il ne sert qu'à conserver le dossier
dans Git/GitHub tant qu'il est vide.
