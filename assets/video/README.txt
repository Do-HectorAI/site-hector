Dossier des vidéos du site Hector AI
====================================

Déposez ici vos vidéos de démonstration (format .mp4 recommandé, sans son).
Fichiers attendus par le site :

- demo-analyse.mp4    → Vidéo de démo de la fonctionnalité « Analyse des dossiers »
                        (page comment-ca-marche.html).

- demo-redaction.mp4  → Vidéo de démo de la fonctionnalité « Rédaction d'actes »
                        (page comment-ca-marche.html).

- hero.mp4            → (Optionnel) Vidéo de fond de la page d'accueil, pour
                        remplacer la photo hero.jpg (voir le commentaire dans
                        index.html, section HERO).

Conseils :
- Les vidéos se lancent automatiquement, en boucle et SANS son (autoplay muted
  loop playsinline). Pour que la lecture automatique fonctionne, la vidéo DOIT
  être muette.
- Compressez vos vidéos (par ex. < 5 Mo) pour un chargement rapide, surtout sur
  GitHub Pages.

Ce fichier README.txt peut être supprimé : il ne sert qu'à conserver le dossier
dans Git/GitHub tant qu'il est vide.
